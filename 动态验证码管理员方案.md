# 管理员动态验证码（TOTP）+ 短期 Token 方案

适用场景：需要一个**简单但安全**的管理员入口，避免固定密码长期有效。

该方案采用：
- **TOTP 动态验证码**（手机验证器每 30 秒变一次 6 位码）
- **短期 JWT Token**（登录后发放，短时有效，过期自动失效）

---

## 1. 总体思路

1. **管理员登录时输入 6 位动态码**  
2. 后端使用 **TOTP Secret** 验证验证码  
3. 验证通过后，后端签发一个 **短期 JWT**（例如 15 分钟）  
4. 管理员接口（删除等敏感操作）必须带 `Authorization: Bearer <token>`  
5. Token 过期后需要重新登录  

这样就能做到：
- 不再使用固定密码
- 即使动态码被截获，也只有 30 秒有效
- Token 失效自动退出管理员

---

## 2. 需要的依赖

后端新增依赖：
- `speakeasy`：TOTP 验证
- `jsonwebtoken`：签发短期 Token

前端不需要新增依赖，只需在登录后保存 token 到 `localStorage`。

---

## 3. 环境变量

后端必须配置：
```
ADMIN_TOTP_SECRET=你的TOTP密钥(Base32)
ADMIN_JWT_SECRET=你的JWT随机密钥
ADMIN_TOKEN_TTL=15m   # 可选，默认 15 分钟
```

### 生成示例（本地）
```
TOTP_SECRET=URHRESLKFEXYROD3XYALPFABEJHMK6V5
JWT_SECRET=7dfbc30ca77dd2a7c8ce5915867603ad4941853f99f591c1b544d8609673cc51
```

TOTP 密钥必须是 **Base32 格式**，JWT 密钥建议使用随机长字符串。

## 4. 管理员登录流程（接口）

### POST `/api/admin/login`

**请求**
```json
{ "code": "123456" }
```

**响应**
```json
{
  "success": true,
  "token": "JWT_TOKEN_STRING",
  "expiresInSeconds": 900
}
```

### 后端伪代码
```js
const verified = speakeasy.totp.verify({
  secret: ADMIN_TOTP_SECRET,
  encoding: 'base32',
  token: code,
  window: 1
});

if (verified) {
  const token = jwt.sign({ role: 'admin' }, ADMIN_JWT_SECRET, { expiresIn: ADMIN_TOKEN_TTL });
  return res.json({ success: true, token });
}
```

## 5. 管理员鉴权中间件

所有敏感接口（如删除记录）都需要验证 token：

```js
const requireAdminAuth = (req, res, next) => {
  const authHeader = req.headers.authorization || '';
  const token = authHeader.startsWith('Bearer ') ? authHeader.slice(7) : '';
  if (!token) return res.status(401).json({ error: 'Missing admin token' });

  try {
    const payload = jwt.verify(token, ADMIN_JWT_SECRET);
    if (payload?.role !== 'admin') return res.status(403).json({ error: 'Forbidden' });
    req.admin = payload;
    next();
  } catch (e) {
    return res.status(401).json({ error: 'Invalid or expired token' });
  }
};
```

使用方式：
```js
app.delete('/api/admin/scores/:id', requireAdminAuth, async (req, res) => {
  ...
});
```

## 6. 前端登录逻辑

1. 管理员输入动态码
2. 调用 `/api/admin/login`
3. 保存 token 到 `localStorage`
4. 后续请求带 `Authorization: Bearer <token>`

### 前端示例
```js
const res = await axios.post('/api/admin/login', { code });
localStorage.setItem('adminToken', res.data.token);
localStorage.setItem('adminTokenExp', String(Date.now() + res.data.expiresInSeconds * 1000));
```

### 删除请求
```js
await axios.delete(`/api/admin/scores/${id}`, {
  headers: { Authorization: `Bearer ${adminToken}` }
});
```

---

## 7. 管理员退出

前端实现“退出管理员”只需要：
```js
localStorage.removeItem('adminToken');
localStorage.removeItem('adminTokenExp');
```

## 8. 如何在手机验证器添加 TOTP

**手动添加**
- 账户名：`项目名-admin`
- 密钥：`ADMIN_TOTP_SECRET`
- 类型：Time-based (TOTP)

或直接生成 URI（二维码也可用）：
```
otpauth://totp/项目名:admin?secret=你的TOTP_SECRET&issuer=项目名&algorithm=SHA1&digits=6&period=30
```

## 9. 常见问题

**Q1：动态码总是验证失败？**  
检查：
- 手机时间是否准确
- `ADMIN_TOTP_SECRET` 是否填错（Base32）
- TOTP 时间窗口是否需放宽（可用 `window: 1`）

**Q2：Token 一直过期？**  
检查：
- `ADMIN_TOKEN_TTL` 是否太短（比如 1m）
- 服务器时间是否正确
